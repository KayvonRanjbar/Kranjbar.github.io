PERSONAL PROJECTS

BrokerViz

https://github.com/Kranjbar/broker-cost


An app for investors who want to easily compare online broker fees personalized to their trading profiles. Users input data about their trading profile, and the app generates a visualization of their projected costs. I conceived the idea and generated wireframes, user stories, and an entity relationship diagram. This was my first time using D3, so I am proud that I was able to effectively utilize this JavaScript library in my project.


Technologies Used

JavaScript, Ruby on Rails, jQuery, D3, PostgreSQL, Bootstrap, HTML/CSS


Sharetastic

https://github.com/kehontas/share


A skills-sharing platform app designed to connect neighbors in a friendly way. Users are able to add skills or services that they are willing to share, and others are able to find skills/services that they are in need of and contact them. Worked in a team of 4 developers in this project. I built out about half the app's features utilizing pair programming. I focused on back-end functionality primarily.


Technologies Used

Ruby on Rails, JavaScript, jQuery, PostgreSQL, bcrypt, Bootstrap, Paperclip, FriendlyId, RSpec, HTML/CSS


ServeNow

https://github.com/Kranjbar/project-01


An interactive single-page community service project app where community organizers can interact with volunteers by posting events that volunteers can sign up to attend. Created application proposal, wireframes, user stories and database diagram; developed both front and back end of app.


Technologies Used

JavaScript, Node.js, jQuery, AJAX, Handlebars, Express, MongoDB, Mongoose, Bootstrap, HTML/CSS
